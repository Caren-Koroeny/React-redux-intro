# 📖 Shopping Cart <a name="about-project"></a>

- Shopping cart items rendered with shopped items . Items can be inceased or decreased in amount and total calculated automatically. Data fetched from an external API using createAsyncThunk.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

- Clone the Repo or Download the Zip file or `https://github.com/Sammy3000/Redux-Intro-Project.git`
- cd /bookstore
- `npm install` To download all the dependancies of the project
- Use `npm start` to run the app in your browser

### Prerequisites

In order to run this project you need:
